Այս նախագծի նպատակն է մեքենայական ուսուցման (Machine Learning) տարբեր ռեգրեսիոն ալգորիթմների միջոցով կանխատեսել տրանսպորտային միջոցների (ավտոմեքենաներ և մոտոցիկլետներ) վաճառքի գինը (Selling_Price) ։ Նախագծում իրականացվել է տվյալների մարզում, մնացորդների (residuals) վերլուծություն, մետրիկաների հաշվարկ և արդյունքների վիզուալիզացիա:  
Տվյալների Բազայի Նկարագրություն
Car_Name — Մեքենայի կամ մոտոցիկլետի մոդելը (օրինակ՝ Swift, Fortuner, Verna, Royal Enfield)
Year — Արտադրության տարեթիվը
Selling_Price — Թիրախային փոփոխական (Target), վաճառքի ընթացիկ գինը
Present_Price — Մեքենայի սրահի (գործարանային) գինը
Kms_Driven — Վազքը (կիլոմետրերով)
Fuel_Type — Վառելիքի տեսակը (Petrol, Diesel, CNG)
Seller_Type — Վաճառողի տեսակը (Dealer, Individual)
Transmission — Փոխանցման տուփը (Manual, Automatic)   
Օգտագործված 7 Մոդելները
Նախագծում կառուցվել և իրար հետ համեմատվել են հետևյալ ռեգրեսիոն մոդելները.
Gradient Boosting Regressor (Գրադիենտային Բուստինգ)
Random Forest Regressor (Պատահական Անտառ)
Decision Tree Regressor (Որոշումների Ծառ)
Weighted KNN Regression (Կշռված K-Մոտակա Դրացիներ)
Standard KNN Regression (Ստանդարտ K-Մոտակա Դրացիներ)
Lasso Regression (Լասսո Ռեգրեսիա)
Linear Regression (Գծային Ռեգրեսիա)
Մոդելների Գնահատման Չափանիշներ (Metrics)
Յուրաքանչյուր մոդելի ճշգրտությունը գնահատվել է 3 հիմնական մետրիկաներով.
MAE (Mean Absolute Error) — Միջին բացարձակ սխալանք
MSE (Mean Squared Error) — Միջին քառակուսային սխալանք
RMSE (Root Mean Squared Error) — Ստանդարտ շեղում (կատարված սխալի քառակուսային արմատը)
 Նախագծի Փուլերը և Կոդի Կառուցվածքը1. Մոդելների Մարզում և ԿանխատեսումՏվյալների նախամշակումից հետո, յուրաքանչյուր մոդել մարզվում է X_train ու y_train բլոկների վրա: Կանխատեսված արժեքները պահվում են առանձին փոփոխականներում (օրինակ՝ predictions_gbr, predictions_rf և այլն)՝ հետագա համեմատության համար։2. Մնացորդների Անալիզ (Residual Analysis)Մոդելների կատարած սխալների բնույթը հասկանալու համար կառուցվել են մնացորդների գրաֆիկներ (Errors = y_test - predictions): Գրաֆիկի վրա $y=0$ հորիզոնական գիծը ցույց է տալիս այն կետերը, որտեղ մոդելը գուշակել է 100% ճշգրտությամբ:3. Համեմատական Բազմապատկեր Գրաֆիկ (Subplots 4x2)Օգտագործված է plt.subplots(nrows=4, ncols=2) ֆունկցիան, որը մեկ ընդհանուր մեծ պատուհանում պատկերում է բոլոր 7 մոդելների Իրական գին ընդդեմ Կանխատեսվածի (Actual vs Predicted) գրաֆիկները։ Կարմիր կետագիծը (y = x) ցույց է տալիս իդեալական կանխատեսման սահմանը:4. Մետրիկաների Ամփոփում և ՎիզուալիզացիաԲոլոր 7 մոդելների հաշվարկված սխալները հավաքվում են միասնական pd.DataFrame-ում: Այնուհետև seaborn.barplot-ի միջոցով կառուցվում են սյունակավոր դիագրամներ (ըստ RMSE և MAE արժեքների), ինչը թույլ է տալիս ակնառու կերպով տեսնել լավագույն արդյունք ցուցաբերած մոդելը:
 Եզրակացություն
Սյունակավոր դիագրամների (Barplots) և մետրիկաների աղյուսակի շնորհիվ հնարավոր է հստակ որոշել, թե որ ալգորիթմն է ապահովում ամենացածր RMSE/MAE արժեքները (այսպիսի տվյալների համար սովորաբար լավագույն արդյունքը ցույց են տալիս Gradient Boosting կամ Random Forest մոդելները), ինչը նշանակում է, որ տվյալ մոդելն ամենահուսալին է մեքենաների շուկայական գները կանխատեսելու համար։
